# Workforce Cost Analysis Using BI

This project focuses on analyzing workforce-related costs using Business Intelligence (BI) tools and interactive dashboards. The main objective is to help organizations understand employee expenses, workforce distribution, productivity, and overall HR performance through data-driven insights.

The BI reports provide a clear visualization of important workforce metrics such as employee salary distribution, department-wise expenses, engagement scores, work-life balance, satisfaction levels, attrition trends, overtime analysis, and workforce demographics. By transforming raw HR data into meaningful dashboards, the project supports better decision-making and workforce planning.

## Key Features

* Workforce cost and salary analysis
* Department-wise employee distribution
* Employee satisfaction and engagement analysis
* Attrition and retention insights
* Work-life balance and overtime tracking
* Interactive dashboards with charts and filters
* Data-driven HR decision support

## Tools & Technologies Used

* Power BI / Business Intelligence Tools
* Microsoft Excel / CSV datasets
* Data Cleaning and Transformation
* Data Visualization Techniques

## Objectives

* To monitor workforce expenses effectively
* To identify trends affecting employee performance and retention
* To improve workforce planning through BI analytics
* To provide interactive and easy-to-understand HR reports

## Outcome

The project demonstrates how Business Intelligence can be used to convert workforce data into actionable insights. These reports help management analyze employee-related costs efficiently and support strategic HR decisions for improving organizational productivity and employee satisfaction.
